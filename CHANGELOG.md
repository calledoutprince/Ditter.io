# Changelog

All notable changes to Ditter.io will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [0.1.0] - 2026-03-01

### Added

- Full canvas functionality with physics engine (`usePhysics` hook)
- Framer-inspired dark UI redesign (top nav bar, left-hand properties sidebar)
- New `Dropdown` component for property controls
- Effect engine via `EffectEngine.jsx`
- Physics element rendering via `PhysicsElement.jsx`
- Integration utilities (`integrations.js`)
- Test suite: unit tests for `EffectEngine`, `PhysicsElement`, and integrations

### Changed

- Renamed project from `temp-vite-app` → `ditter-io` in `package.json`
- Major UI/UX overhaul — layout, typography, color system updated to dark theme

### Fixed

- Various UI/UX polish fixes
